# ClassManager Pro

מערכת ניהול פדגוגית הוליסטית למורים, المבוססת על בינה מלאכותית (AI). 
המערכת הורחבה ממיקוד בלעדי של התאמת מקומות ישיבה, למערכת CRM פדגוגית שלמה שנותנת למורה כלי עבודה עוצמתיים לניהול הכיתה ביומיום.

## תכונות מרכזיות

* **דשבורד פדגוגי מיחידי (Dashboard)**: תצוגה מרכזית של מצב הכיתה, נוכחות יומית, סטטיסטיקות תלמידים וציונים.
* **ניהול תלמידים מתקדם (Student CRM)**: פרופיל מקיף לכל תלמיד הכולל:
  * מידע חברתי ולמידה
  * אבחונים והתאמות אישיות
  * מעקב הישגים וציונים (Grades & Progress)
  * היסטוריית נוכחות (Attendance)
  * תיעוד תקשורת מול הורים וגורמים טיפוליים
  * המלצות פדגוגיות מבוססות AI לכל תלמיד
* **סידור ישיבה חכם (Smart Seating)**: אלגוריתם מתקדם (Simulated Annealing) לשיבוץ תלמידים במרחב הלמידה, תוך התחשבות באילוצים חברתיים (חברים, הפרדות), אילוצים מרחביים (מרחק, גובה) ופיזיות של הכיתה (עמודים, שולחנות ורווחים).
* **עוזר AI אוטומטי (Floating AI)**: ממשק הפעלה קולית וטקסטואלית המאפשר לבקש פעולות בשפה טבעית (למשל "הוסף לתמר ציון 90 במתמטיקה" או "המקום של דני רועש מדי, נסה למצוא לו מקום חלופי").

## חוויית משתמש ועיצוב

ממשק המשתמש עוצב מחדש כדי להעמיד את ניהול הפדגוגיה והמידע במרכז:
* **תפריט צד עשיר**: מעבר מהיר בין ניהול נוכחות, ציונים, אבחונים, תכנון כיתה ודוחות.
* **מסך פתיחה (סקירה)**: הצגת נתונים מסכמים כבר עם הכניסה למערכת.
* **תצוגות מתחלפות למרחב הפיזי**: בין מעקב נוכחות במבט כיתתי לבין מבט מבחנים או מצב למידה כללי, הכל על בסיס מפת הושבה עדכנית.

## טכנולוגיה

המערכת פותחה באמצעות React, TypeScript, Tailwind CSS, ומשתמשת ב-Google Gemini API לניתוח שפה טבעית ואלגוריתמיקה חכמה. השמירה מתבצעת באופן לוקאלי בדפדפן (LocalStorage) עם אפשרות לייצוא וייבוא מלא של הנתונים כדי למנוע אובדן מידע.
