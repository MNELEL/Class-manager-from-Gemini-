# ClassPro Pro-Manager

**ClassPro Pro-Manager** היא מערכת פדגוגית הוליסטית מתקדמת למורים ומחנכים, המשלבת בינה מלאכותית (AI) כדי להפוך את ניהול הכיתה לחכם, יעיל ומרגש יותר. המערכת מעניקה למורה שליטה מלאה בנתונים הפדגוגיים, החברתיים והלוגיסטיים של הכיתה, ומאפשרת להתמקד בעיקר: בחינוך ובלמידה.

## 🎯 מטרות המערכת
*   **אופטימיזציה פדגוגית:** יצירת סביבת למידה המותאמת לצרכים הייחודיים של כל תלמיד ותלמידה.
*   **ניהול נתונים מרכזי (Single Source of Truth):** ריכוז ציונים, נוכחות, הערות והתאמות במקום אחד נגיש ומאובטח.
*   **אוטומציה חכמה:** צמצום העומס הבירוקרטי על המורה באמצעות כלי AI לשיבוץ וקיבוץ תלמידים.

## 🚀 פיצ'רים מרכזיים

### 1. דירוג וסידור ישיבה חכם (AI Smart Sort)
*   **אלגוריתם סידור אוטומטי:** שיבוץ תלמידים במרחב בהתחשב במאות פרמטרים (גובה, קשיים לימודיים, מוקדי רעש).
*   **ניהול אילוצים מתקדם:** הגדרת תלמידים שחייבים לשבת יחד, תלמידים שחייבים להיפרד, והתחשבות בקרבה ללוח או לפתח.
*   **עורך כיתה אינטראקטיבי:** גרירה ושחרור (Drag & Drop) של תלמידים ושולחנות, הגדרת עמודים ומעברים במרחב.
*   **תצוגת 3D:** הדמיה תלת-ממדית של הכיתה לקבלת תחושה אמיתית של המרחב.

### 2. ניהול קבוצות ואילוצים (Group Management)
*   **ממשק ניהול ייעודי:** יצירת קבוצות למידה קבועות או משתנות.
*   **אילוצים חברתיים:** הגדרה מפורטת של יחסים בין תלמידים שה-AI מתחשב בהם בכל סידור מחדש.

### 3. ניהול תלמידים (Pedagogical CRM)
*   **כרטיס תלמיד מורחב:** תיעוד מלא הכולל כינויים, רמת עניין, צורך בתמיכה, הערות ותחביבים.
*   **מעקב ציונים והישגים:** יומן ציונים מפורט עם הצגת ממוצעים ומגמות התקדמות.
*   **יומן נוכחות:** מעקב יומי קל לניווט אחר נוכחות, איחורים וחיסורים.

### 4. כלי שיתוף וייצוא נתונים
*   **Excel & CSV Support:** ייבוא מהיר של רשימות תלמידים מקבצי Excel קיימים או CSV.
*   **הפקת דוחות:** ייצוא נתוני הכיתה והציונים לפורמטים של Excel, PDF המעוצב להדפסה, או JSON לגיבוי.
*   **ייצוא ויזואלי:** שמירת סידור הישיבה (Layout) כקובץ תמונה (PNG) או מסמך PDF לשיתוף מהיר והדפסה.

### 5. ניהול מבצעים ואתגרים (Classroom Campaigns)
*   **מנוע מבצעים חכם:** יצירת אתגרים כיתתיים (משותפים) או אישיים (תחרותיים).
*   **מעקב התקדמות ויזואלי:** גרפי התקדמות בזמן אמת המציגים את עמידת התלמידים ביעדי המבצע.
*   **מסך הקרנה (Kiosk Mode):** תצוגה ייעודית להקרנה בכיתה המציגה את מצב המבצע בדומה ללוח תוצאות משחקי.

### 6. עיצוב והתאמה אישית (Classroom Themes)
*   **ערכות נושא:** בחירה בין מגוון חוויות סביבתיות (טבע, יער, אוקיינוס, שקיעה, מלכותי) המשפיעות על נראות הכיתה בתלת-מימד.

### 7. ארגז כלים וניהול זמן (Teacher Toolkit)
*   **גלגל שמות (Winner Wheel):** בחירת תלמיד אקראית למשימה או תשובה בצורה חווייתית.
*   **מחולל קבוצות אקראי:** חלוקת הכיתה לקבוצות עבודה בשניות.
*   **יומן אירועים ותיעוד:** תיעוד אירועים כיתתיים חשובים, טיולים, מבצעים מיוחדים ומבחנים עם תיאור מפורט.
*   **מערכת תזכורות:** ניהול תזכורות למשימות דחופות, פגישות הורים או הכנת שיעורים, כולל התראות ויזואליות.

## 🛠 טכנולוגיה
המערכת בנויה בטכנולוגיות המתקדמות ביותר:
*   **Framework:** React 18 עם TypeScript ליציבות מקסימלית.
*   **Styling:** Tailwind CSS לעיצוב מודרני, נקי ורספונסיבי.
*   **Animations:** Framer Motion לחוויית משתמש חלקה.
*   **Artificial Intelligence:** מנוע Gemini AI לניתוח טקסטואלי ואופטימיזציה.
*   **Data Handling:** שימוש בספריית `xlsx` לעבודה עם גליונות נתונים וב-`jsPDF` להפקת דוחות.
*   **Storage:** השמירה מקומית (LocalStorage) עם הצפנה בסיסית, השומרת על פרטיות התלמידים במכשיר המורה בלבד.

## 🌈 חווית משתמש
*   **Dark Mode:** תמיכה מלאה במצב כהה לעבודה נוחה בכל שעות היממה.
*   **RTL Support:** תמיכה מלאה בעברית וביישור מימין לשמאל.
*   **Mobile Ready:** האפליקציה רספונסיבית לחלוטין ועובדת נהדר גם בסמארטפונים ובטאבלטים.
