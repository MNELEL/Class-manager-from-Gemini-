# ClassPro Pro-Manager

**ClassPro Pro-Manager** היא מערכת פדגוגית הוליסטית מתקדמת למורים ומחנכים, המשלבת בינה מלאכותית (AI) כדי להפוך את ניהול הכיתה לחכם, יעיל ומרגש יותר. המערכת מעניקה למורה שליטה מלאה בנתונים הפדגוגיים, החברתיים והלוגיסטיים של הכיתה.

## 🎯 מטרות המערכת
*   **אופטימיזציה פדגוגית:** התאמת סביבת הלמידה לצרכים הייחודיים של כל תלמיד.
*   **ניהול נתונים מרכזי:** ריכוז כל המידע הפדגוגי במקום אחד נגיש ומאובטח.
*   **אוטומציה חכמה:** צמצום עומס בירוקרטי בעזרת כלי AI למשימות שגרתיות.

## 🚀 פיצ'רים קיימים (מערכת פעילה)

### 1. דירוג וסידור ישיבה חכם (AI Smart Sort)
*   **מה זה:** אלגוריתם מתקדם לשיבוץ תלמידים לפי מאות פרמטרים (צרכים לימודיים, חברתיים, תנאי כיתה).
*   **תועלת:** חיסכון עצום בזמן למורה, יצירת סביבה שתומכת באקלים כיתתי חיובי והפחתת קונפליקטים.

### 2. ממשק עריכה ויזואלי
*   **מה זה:** עורך כיתה אינטראקטיבי מבוסס Drag & Drop, תמיכה בתצוגת רשת ותלת-מימד בסיסית.
*   **תועלת:** שליטה מוחלטת בסידור הריהוט והתלמידים, הבנה מרחבית טובה יותר של הכיתה.

### 3. ניהול תלמידים והישגים (Pedagogical CRM)
*   **מה זה:** תיעוד מלא: כרטיס תלמיד, יומן ציונים, נוכחות, הערות והתאמות.
*   **תועלת:** ראייה הוליסטית של התלמיד, קבלת החלטות מבוססת נתונים (Data-Driven).

### 4. ניהול מבצעים ואתגרים (Gamification)
*   **מה זה:** מנוע מבצעים כיתתיים, מעקב התקדמות ויזואלי, ומסך הקרנה חווייתי (Kiosk Mode).
*   **תועלת:** הגברת מוטיבציה, עידוד התנהגות חיובית, הפיכת הלמידה למשחק חיובי.

### 5. ארגז כלים וניהול זמן (Teacher Toolkit)
*   **מה זה:** גלגל שמות, מחולל קבוצות, יומן אירועים, ותזכורות.
*   **תועלת:** שדרוג הפעילות השוטפת בכיתה והורדת עומס ניהולי.

## 🗺️ מפת דרכים (פיצ'רים בתכנון)

| פיצ'ר | תיאור | תועלת צפויה |
| :--- | :--- | :--- |
| **מצב מצגת (Presentation Mode)** | הסתרת UI ומעבר למבט 3D נקי | מיקוד מקסימלי בזמן הצגה ללא הפרעות |
| **סנכרון Google Calendar** | חיבור ישיר ללוח השנה האישי | ניהול זמן מאוחד ומניעת התנגשויות |
| **שיפור עריכה ויזואלית** | התאמות spacing, נעילת שולחנות ו ghost-drag | דיוק גבוה יותר בסידור, נוחות שימוש משופרת |
| **Toolbar מתקדם** | Undo/Redo וחיפוש מהיר | מניעת טעויות והגברת יעילות בניהול |
| **מערכת התראות חכמה** | התראות על דדליינים, ימי הולדת וקונפליקטים | מניעת שכחה, מענה אקטיבי לצרכי תלמידים |
| **דף נחיתה אינטראקטיבי** | הסבר ויזואלי על תכונות ה-AI | הצגת הערך המוסף למשתמשים חדשים |

## 🛠 טכנולוגיה
המערכת בנויה על: React, TypeScript, Tailwind CSS, Framer Motion, ו-Gemini AI.
